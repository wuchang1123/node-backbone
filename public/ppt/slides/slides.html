产品开发技术分享

Node.js & Backbone.js

高仿真环境开发

  1. 本地创建站点文件夹
  2. 架设服务器 + host 指向
  3. 模拟服务端处理、返回html/json/其他内容(php,nodejs)
  4. 使用编辑器的站点功能管理、编辑代码
  5. 编译（less、模板预编译）、发布

高仿真环境好处

  1. 不需等待后端开发的接口
  2. 域名绑定后还可以直接使用到线上的cookie、避开ajax跨域等问题
  3. PC端主机，其他人可以方便访问确认效果
  4. 可以方便控制返回的数据，查看页面是否破版

Node.js

前端方面可使用的特征、功能

  • 内核 V8 JavaScript Engine
  • 模块划分功能，包管理 NPM，大量开源功能代码
  • 编译（less、模板预编译）
  • 发布命令 (Grunt)

Node.js

对前端好处

  • 能简单架设本地服务器，不需要学习服务器配置和其他编程语言
  • 友好的网址（类似 apache rewrite）
  • 编译功能，可以使得css、js可以按照自己想的方式编写

安装

  • Node.js 有各个系统的安装包 下载
  • npm install 模块名字 [-g]
    • 模块安装，当前项目可以引用
    • 可选项 -g 为全局安装，所有项目都可以直接引用

less

  • LESS 将 CSS 赋予了动态语言的特性，如 变量， 继承， 运算， 函数. LESS 既可以在 客户端 上运行，也可以借助Node.js/php在服务端运行。
  • 建议开发环境使用服务端模式，上线前使用nodejs编译成css发布
